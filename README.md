# Coved19_Data_Analysis

Short project for analyzing COVID-19 data using a Jupyter Notebook.

## Overview
This repository contains a notebook (src/Covid19_Data_Analysis.ipynb) that loads COVID-19 data, performs basic cleaning, aggregates by country, and provides visualizations of confirmed, recovered, deaths, and active cases.

## Data
The notebook reads data from the data folder (data/Covid19_Data.xls).

## Requirements
- Python 3.x
- pandas
- numpy
- matplotlib
- seaborn

Install dependencies with:

pip install -r requirements.txt

(If no requirements.txt is provided, install packages manually.)

## Usage
Open and run the notebook in Jupyter:

jupyter notebook src/Covid19_Data_Analysis.ipynb
